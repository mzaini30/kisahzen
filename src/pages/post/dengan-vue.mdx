---

Dengan menggunakan Vue, kamu bisa membuat banyak hal. Mulai dari website SPA, website blog, aplikasi Android, Chrome extension. Dan masih banyak hal lain yang bisa kamu buat menggunakan Vue.

Kalau untuk membuat website blog, bisa menggunakan Iles yang saat ini sedang aku gunakan. Dia itu SSG. Cocok bagi blog karena nggak perlu root hydration. Dia pakainya partial hydration.

Kok bisa sih membuat banyak web apps itu sih? 

Ya karena memang yang namanya Vue ini adalah tools untuk mengatur tampilan. Ya sama dengan React, Angular, dan Svelte. Jadi, nggak bisa sih kita mengatakan bahwa framework A lebih unggul daripada framework B, framework B lebih unggul daripada framework C. Kok bisa? Karena memang masing-masing itu unggul dengan kemampuannya masing-masing.

Jadi, di sini aku memilih Vue daripada React maupun Svelte ya karena pengen aja. Dan memang alhamdulillah bisa dengan mudah diintegrasikan dengan tools lain sehingga bisa membuat berbagai jenis aplikasi dengan sangat mudah.

Coba deh aku list ulang tadi yang bisa dibuat dengan Vue:

- Website SPA
- Website MPA
- Aplikasi Android
- Chrome extension
- Firefox addons
- Template Blogspot (WIP)

Kalau website SPA, insya Allah sudah paham semua kan ya. Jadi, website SPA (single page application) itu nanti hanya memerlukan sebongkah file index.html yang ditemani oleh banyak file CSS dan JS. Cara ini cukup simpel dan sangat familiar di kalangan programmer frontend dalam membuat tampilan karena konfigurasinya yang sangat mudah.

Kemudian, ada website MPA. Di sini, fokusnya adalah SEO. Jadi, setiap halaman ada file HTML-nya masing-masing.
