---
title: Vercel jadi Backend JavaScript
description: Menggunakan Vercel sebagai backend dengan JavaScript
date: 2020-06-05
---

Isi dari `vercel.json`:

```json
{
  "version": 2,
  "builds": [
    {
      "src": "./index.js",
      "use": "@vercel/node"
    }
  ],
  "routes": [
    {
      "src": "/(.*)",
      "dest": "/"
    }
  ]
}
```
